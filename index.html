Dont' Miss Out on the Next Big Event!

This web application is your one stop shop for important information. Research the ticketable events taking place near you or for an upcoming trip. Whether it's a game or concert get the details you need. Plan ahead for a fun worry free time.

This web app is styled with Foundation by Zurb

A framework for any device, medium, and accessibility, Foundation is a family of responsive front-end frameworks that make it easy to design beautiful responsive websites, apps and emails that look amazing on any device.

Foundation is semantic, readable, flexible, and completely customizable. The Yeti, Snowman, and Squid are constantly adding new resources and code snippets to help get you started! ;)
